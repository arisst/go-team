# Installation
- Clone this repository
- Adjust config file with your environment in `.env` file
- Run `go run main.go`
- Test `go test -v`

# Or with Docker Compose
- Run `docker-compose up -d`
- Test `docker-compose exec app go test -v`
- Goto `http://localhost:8000/api`
